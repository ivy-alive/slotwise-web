# SlotWise Web

Frontend for Slotwise, a personal scheduling system for study and workout tasks.

## Tech Stack

- React 18
- Vite
- shadcn/ui
- Tailwind CSS
- Axios

## Getting Started

### Prerequisities

- Node.js 20+

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

Visit http://localhost:5173
